

同步函数里面，可以有异步函数async，但是由于外层是同步函数，所以不能用await，必须得用asyncio.run(async_fun1())本质上是顺序执行
异步函数里面，可以有同步函数，也可以有异步函数，如果外层是异步函数，则必须用await，不能用asyncio.run(sync_fun1())


奇怪的是这个异步函数里面的同步简单操作，比如加法啥的会顺序执行的
```python
import asyncio
import time

def sync_fun1():
    print("sync_fun1")
    time.sleep(1)
    a=1
    b=2
    c=3
    a=a+b
    b=b+c
    c=a+b
    print(a)
    print(b)
    print(c)
    print("sync_fun1 end")
    return 1

def sync_fun2():
    print("sync_fun2")
    time.sleep(1)
    print("sync_fun2 end")
    return 2


async def main():
    a=sync_fun1()
    print(a)
    b=sync_fun2()
    print(b)


if __name__ == "__main__":
    asyncio.run(main())
```
